\chapter*{Аннотация}
\addcontentsline{toc}{chapter}{Аннотация}

В данном программном документе приведено руководство программиста по
настройке и использованию программы <<\programtitle>>, предназначенной для визуализации эквипотенциальных поверхностей и силовых линей двух полей, заданных двумя векторами и областью определения, а также расчета и отображения параметров этих вектор-полей.

В данном программном документе, в разделе <<Назначение и условия
применения программы>> указаны назначение и функции, выполняемые программой,
условия, необходимые для выполнения программы (объем оперативной памяти,
требования к составу и параметрам периферийных устройств, требования к
программному обеспечению и т.п.).

В разделе <<Характеристика программы>> приведено описание основных
характеристик и особенностей программы (режим работы, средства контроля
правильности выполнения и т.п.).

В данном программном документе, в разделе «Входные и выходные данные»
приведено описание организации используемой входной и выходной информации.

В разделе <<Сообщения>> указаны тексты сообщений, выдаваемых
программисту или оператору в ходе выполнения программы, описание их
содержания и действий, которые необходимо предпринять по этим сообщениям.

\chapter{Назначение и условие применения программы}
\section{Назначение программы}

Программа предназначена для визуализации эквипотенциальных полей и силовых линий, заданных двумя векторами и областью определения, а также расчета и отображения параметров этих вектор-полей.

\section{Функции, выполняемые программой}

Функциями программы являются:

\begin{enumerate}[label=\asbuk*),ref=\asbuk*]
	\item вычисление потока поля \( \vec a \) через поверхность \( S \) (поверхность \( S \) --- часть единичной сферы \(x^2 + y^2 + z^2 = 1 \), лежащая в первом октанте (\( x \geq 0 \), \( y \geq 0 \), \( z \geq 0 \)) в направлении внешней нормали для сферы двумя способами:
	\begin{enumerate}[label=\arabic*.]
		\item по формуле Гаусса-Остроградского, замкнув поверхность \( S \) частями координатных плоскостей;
		\item с помощью поверхностного интеграла;
	\end{enumerate}
	
	\item вычисление циркуляции \( C \) поля \( \vec a \) по контуру поверхности \( S \) в положительном направлении со стороны внешней нормали двумя способами:
	\begin{enumerate}[label=\arabic*.]
		\item по формуле Стокса;
		\item с помощью криволинейного интеграла;
	\end{enumerate}
	
	\item изучение поля \( \vec b \):
	\begin{enumerate}[label=\arabic*.]
		\item проверить потенциальность поля \( b \), убедившись, что \( \mathrm{rot} \, \vec b = 0 \);
		\item вычислить потенциал \( \varphi \) поля \( b \) по формуле \[ \varphi = \int_0^x{b_x(x,0,0)\,dx} + \int_0^y{b_y(x,y,0)\,dy} +\int_0^z{b_z(x,y,z)\,dz}\;; \]
		\item проверить безошибочность нахождения потенциала с помощью формул \[ \varphi'_x = b_x\mathrm{,}\;\varphi'_y = b_y\mathrm{,}\;\varphi'_z = b_z\;; \]
		\item вычислить с помощью потенциала \( \varphi \) работу \( W \) сил поля по перемещению материальной точки из точки \(O(0,0,0) \) в точку \( M(1,1,1) \), убедиться, что ответ \( W = 3 \) (контрольное число);
	\end{enumerate}
	
	\item визуализация векторных полей: построение и отображение силовых линий поля \fieldA, построение и отображение силовых линий поля \fieldB;
	
	\item визуализация эквипотенциальных поверхностей: вычисление и отображение эквипотенциальных поверхностей для поля \fieldA, вычисление и отображение эквипотенциальных поверхностей для поля \fieldB.
\end{enumerate}

\section{Условия, необходимые для выполнения программы}

Для обеспечения функционирования программы на компьютере, на котором будет эксплуатироваться данный программный продукт, должно быть установлено следующее программное обеспечение:

Наименование программы: C++.

Обозначение программы: C++.

Версия программного изделия: 15.0

\subsection{Объем оперативной памяти}

Оперативная память объемом не менее 256 Мб.

\subsection{Требования к составу периферийных устройств}

Персональный компьютер, на котором выполняется программа должен включать в свой состав:

\begin{itemize}
	\item манипулятор типа <<мышь>>;
	\item клавиатура.
\end{itemize}

\subsection{Требования к параметрам периферийных устройств}

Специальных требований к параметру периферийных устройств не предъявляется.

\subsection{Требования к программному обеспечению}

Для обеспечения функционирования программы на компьютере, на котором будет эксплуатироваться данный программный продукт, должно быть установлено следующее программное обеспечение:

Наименование программы: Visual Studio.

Обозначение программы: Visual Studio.

Версия программного изделия: 2019 и выше.

\section{Требования к персоналу (программисту)}

Программист должен иметь минимум среднее техническое образование и должен быть аттестован минимум на I квалификационную группу по электробезопасности.

В перечень задач, выполняемых программистом, должны входить:

\begin{enumerate}
	\item[а)] Задача поддержания работоспособности технических средств;
	\item[б)] Задача поддержания работоспособности системных программных средств -- операционной системы;
	\item[в)] Задача поддержания работоспособности программы \programtitle.
\end{enumerate}

\section{Организация используемой входной информации}
Входные данные могут быть представлены (по выбору пользователя):
\begin{enumerate}[label=\asbuk*),ref=\asbuk*]
	\item в виде данных, введенных пользователем с клавиатуры;
	\item в виде данных, хранящихся в файле формата \texttt{.txt}.
\end{enumerate}

Входной файл формата \texttt{.txt} содержит:
\begin{enumerate}[label=\arabic*.,ref=\arabic*]
